# 🗳️ Political & Election Analysis

## Machine Learning Based Election Winner Prediction Dashboard

A complete Data Science and Machine Learning project that analyzes historical Indian Assembly Election data and predicts winning political parties using Machine Learning.

🌐 **Live Demo**

https://political-election-analysis-vhvvyqb8uiobvrtx7ujkmx.streamlit.app

---

## 📌 Project Overview

This project performs an in-depth analysis of historical election data and predicts winning political parties using Machine Learning algorithms.

The dashboard allows users to:

- 📊 Explore historical election trends
- 🏛 Analyze constituency-wise election data
- 🔮 Predict winning political parties
- 🚀 Simulate future election predictions
- 📈 Compare Machine Learning models

---

## ✨ Features

### 📊 Election Analysis Dashboard

- State-wise Analysis
- Party-wise Performance
- Election Trends
- Interactive Charts

### 🔮 Winner Prediction

- Historical Winner Prediction
- Future Election Simulation
- Prediction Confidence
- Candidate Information

### 🤖 Machine Learning Models

The following algorithms were trained and evaluated:

- 🌳 Decision Tree
- 🌲 Random Forest
- 🌳 Extra Trees

The **Decision Tree Classifier** was selected for deployment because it provides:

- ✅ Highest Accuracy (98.71%)
- ⚡ Fast Prediction
- 💾 Small Model Size
- 🌐 Efficient Streamlit Deployment

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Abhinav-T9947/Political-Election-Analysis.git
```

Move into the project directory

```bash
cd Political-Election-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
